Frontend Architecture

Aplikasi web dibangun sebagai Single Page Application (SPA) menggunakan teknologi modern.

Core Tech Stack

  • Framework: React v19 + Vite (Build Tool & Dev Server).
  • Styling: Tailwind CSS v4 (Utility-first) + Basecoat CSS.
  • Animation: Framer Motion 12 (untuk transisi halaman dan elemen UI).
  • Routing: React Router DOM v7.
  • State Management: React Context API (ChatContext, UserContext, AuthContext).

Key Libraries

Library Version Usage
axios ^1.13.2 HTTP Client untuk komunikasi ke Backend API.
lucide-react ^0.563.0 Koleksi icon SVG yang ringan dan konsisten.
qrcode.react ^4.2.0 Generate QR Code untuk kartu member digital.
react-markdown ^10.1.0 Rendering format markdown pada pesan chat AI.

Project Structure

src/
├── components/   # Reusable UI components (Button, Modal, Input)
├── context/      # Global state (Auth, Chat, User)
├── hooks/        # Custom React Hooks (useAuth, useChat)
├── layouts/      # Page layouts (UserLayout, MainLayout)
├── pages/        # Route page components
│   ├── admin/    # Admin dashboard pages
│   ├── owner/    # Store owner dashboard pages
│   └── user/     # Customer facing pages
├── services/     # API service functions (centralized Axios calls)
└── styles/       # Global CSS & Tailwind directives

Backend Architecture

Backend dibangun menggunakan Node.js (TypeScript) dengan pendekatan Modular Monolith.

Core Tech Stack

  • Runtime: Node.js v22 (Latest LTS).
  • Framework: Express.js v5. Beta version untuk performa lebih baik.
  • Database: PostgreSQL 16.
  • ORM: Prisma v6. Menggunakan schema-first approach.
  • AI Engine: Google Generative AI (Gemini 1.5 Flash).

Architectural Pattern

Project ini mengikuti Controller-Service-Repository pattern namun disederhanakan menjadi Modular:

  • Modules: Setiap fitur (Auth, Chat, Product) memiliki foldernya sendiri yang berisi Route, Controller, dan Schema.
  • Common: Folder src/common berisi shared logic seperti Middleware (Auth, RBAC), Utilities, dan Config.

Installation Guide

Sistem ini membutuhkan Node.js v18+ dan PostgreSQL.

Backend Setup

cd backend
npm install
cp .env.example .env
npx prisma generate
npx prisma db push
npm run dev

Frontend Setup

cd frontend
pnpm install
pnpm run dev

Authentication Module

Standar JWT. Header wajib: Authorization: Bearer .

Register User

POST /api/auth/register
Field Type Required Description
name String Yes Nama lengkap.
email String Yes Email unik.
password String Yes Min 8 karakter.
role String Yes USER atau OWNER.

Login

POST /api/auth/login
Field Type Required Description
email String Yes Email user.
password String Yes Password user.

Update Profile

PATCH /api/auth/profile Auth
Field Type Required Description
name String No Update nama.
phone String No Update kontak.
currentPassword String Cond Wajib jika ubah password.

Chat & AI Service

Modul interaksi utama antara User dan Sistem/Store.

Send Message (AI)

POST /api/chat Auth

Mengirim pesan ke AI Gemini untuk mendapatkan respons produk.

Field Type Required Description
message String Yes Pesan user.
ownerId UUID Yes ID Toko.
sessionId UUID No ID Sesi percakapan.

Call Staff

POST /api/chat/call-staff Auth

Memanggil staff toko untuk bantuan langsung (Live Support).

Field Type Required Description
ownerId UUID Yes Toko tujuan.
latitude Float No Lokasi user.
longitude Float No Lokasi user.

Stop Staff Call

POST /api/chat/stop-staff Auth

Mengakhiri sesi live support dari sisi user.

Manage Sessions

POST /api/chat/sessions Auth

Membuat sesi chat baru secara manual.

GET /api/chat/sessions/:sessionId/messages Auth

Mengambil riwayat pesan dari sesi spesifik.

DELETE /api/chat/sessions/:sessionId Auth

Menghapus sesi chat tertentu.

Chat History

GET /api/chat/history Auth

Mengambil seluruh riwayat chat user.

DELETE /api/chat/history Auth

Menghapus seluruh riwayat chat.

Staff Actions (Owner Only)

POST /api/chat/accept-call Owner

Menerima panggilan bantuan dari user.

POST /api/chat/decline-call Owner

Menolak panggilan bantuan.
